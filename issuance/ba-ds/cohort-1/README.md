## BA/DS Cohort 1

Recipients for the BA/DS program Cohort 1 are listed in `recipients.csv`.

Before issuing/exporting certificates, fill in:

- `cert_id`
- `verify_url`
- `issue_date` (if different from the template default)
- `duration` (if different from the template default)

Store exported files in `exports/`.

Export folders created:

- `exports/png`
- `exports/svg`
- `exports/html`

Bitly placeholders are listed in `links/bitly-placeholders.csv`.
